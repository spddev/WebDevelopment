Игры и сервисы для обучения программированию

Screeps — первая в мире стратегия MMO для программистов

Это игра с открытым исходным кодом для программистов, в которой основной механикой является программирование ИИ ваших юнитов.
Вы управляете своей колонией, написав код на JavaScript.
Настоящее программирование, а не просто псевдокод.
Вы можете использовать библиотеки из реальных проектов.
Кластер из 40 серверов (160 ядер ЦП), обрабатывающий сценарии проигрывателя с помощью Node.js
Огромный постоянный мир, состоящий из 70 000 соединенных между собой игровых комнат.

Katacoda — интерактивная обучающая платформа для разработчиков

Это платформа для создания интерактивных демонстрационных и обучающих сред в реальном времени. Среды можно настроить в соответствии с требованиями ваших приложений.
Более 250 бесплатных интерактивных сценариев обучения, помогающих решать реальные проблемы и приобретать новые навыки.
Лучший способ учиться - это делать.
Все учебные пособия интерактивны с предварительно настроенными живыми средами, готовыми к использованию.

w3schools — бесплатный сервис для изучения Python, Java и других языков

Для любителей разработки это отличный ресурс, который позволяет проверить ваши знания. На сайте можно изучить теоретические основы языка и примеры его использования, а также поупражняться в создании программ по всем представленным материалам. Это бесплатные учебные пособия и справочные материалы по всем темам веб-разработки, таким как HTML, CSS, JavaScript, Python, PHP, SQL, Java и многое другое. Еще вы можете поиграть в кодовую игру: Помогите рыси собрать шишки!
